## Setup
- Add `xui-module` dependency using yarn or npm to your project
- Add `xui-module` to `modules` section of `nuxt.config.js`

```js
{
  modules: [
    // Simple usage
    'xui-module',

    // With options
    ['xui-module', { /* module options */ }],
 ]
}
```
